# ウィンドウ
{app}は特定のタスクに個別のウィンドウを使用します。各ウィンドウはトレイコンテキストメニューまたはグローバルキーボードショートカットから開けます。

## 🏷  ラベルウィンドウ
トレイメニュー、グローバルショートカット、またはメインウィンドウのタグボタンから開きます。現在のEEGモーメントにアノテーションを付けるフリーテキストラベルを入力します（例：「瞑想」、「集中読書」）。ラベルは正確なタイムスタンプ範囲とともに{dataDir}/labels.sqliteに保存されます。Ctrl/⌘+Enterまたは送信ボタンで送信します。Escapeでキャンセルします。

## 🔍  検索ウィンドウ
検索ウィンドウにはEEG類似検索、テキスト検索、インタラクティブ検索の3つのモードがあり、それぞれ異なる方法で記録データをクエリします。

## EEG類似検索
開始/終了の日時範囲を選択し、そのウィンドウ内で記録されたすべてのEEG埋め込みに対して近似最近傍検索を実行します。HNSWインデックスは全履歴からコサイン距離で最も類似したk個の5秒EEGエポックを返します。距離が低いほど脳状態が類似しています。結果のタイムスタンプに重なるラベルがインラインで表示されます。参照期間と「感覚が似ている」過去の瞬間を見つけるのに役立ちます。

## テキスト埋め込み検索
任意の概念、アクティビティ、または精神状態を自然言語で入力します（例：「deep focus」、「anxious」、「eyes closed meditation」）。クエリはラベルインデックスに使用されるのと同じsentence-transformerモデルで埋め込まれ、HNSWラベルインデックス上のコサイン類似度を使用して、過去に作成したすべてのアノテーションとマッチングされます。結果はキーワードマッチングではなく、セマンティック類似度でランク付けされた独自のラベルです。リストのフィルタリングや日付・類似度での再ソートが可能です。3D kNNグラフが近傍構造を可視化します: クエリノードが中心に位置し、結果ラベルが距離に応じて放射状に配置されます。

## インタラクティブ・クロスモーダル検索
フリーテキストの概念を入力すると、{app}は4段階のクロスモーダルパイプラインを実行します: (1) クエリをテキストベクトルに埋め込み、(2) セマンティックに最も類似したk個のラベルを取得（text-k）、(3) マッチした各ラベルについて平均EEG埋め込みを計算し、日次EEG HNSWインデックスから最も類似したk個のEEGモーメントを検索（eeg-k）、(4) 各EEG近傍について±reach分以内の近傍ラベルを収集（label-k）。結果は4つのノードレイヤー（クエリ → テキストマッチ → EEG近傍 → 発見ラベル）を持つ有向グラフとしてレンダリングされ、インタラクティブな3Dビジュアライゼーションとして表示されます。SVGまたはGraphviz DOTとしてエクスポートできます。text-k / eeg-k / label-kスライダーでグラフ密度を制御し、±reachで時間的検索ウィンドウを広げたり狭めたりします。

## 🎯  キャリブレーションウィンドウ
ガイド付きキャリブレーションタスクを実行します: アクションフェーズの交互実行（例：「開眼」→ 休憩 →「閉眼」→ 休憩）を設定可能なループ回数で行います。接続済みでストリーミング中のBCIデバイスが必要です。キャリブレーションイベントはTauriイベントバスとWebSocket経由で発信されるため、外部ツールが同期できます。最後に完了したキャリブレーションのタイムスタンプは設定に保存されます。

## ⚙  設定ウィンドウ
4つのタブがあります: 設定、ショートカット（グローバルホットキー、コマンドパレット、アプリ内キー）、EEGモデル（エンコーダーとHNSWステータス）。トレイメニューまたはメインウィンドウのギアボタンから開きます。

## ?  ヘルプウィンドウ
このウィンドウです。{app}インターフェースのすべての部分 — メインダッシュボード、各設定タブ、各ポップアップウィンドウ、トレイアイコン、WebSocket API — の完全なリファレンスです。トレイメニューから開きます。

## 🧭  セットアップウィザード
Bluetoothペアリング、ヘッドセットの装着、最初のキャリブレーションをガイドする5ステップの初回起動ウィザードです。初回起動時に自動的に開きます。コマンドパレット（⌘K → セットアップウィザード）からいつでも再度開けます。

## 🌐  APIステータスウィンドウ
現在接続中のすべてのWebSocketクライアントとスクロール可能なリクエストログを表示するライブダッシュボードです。サーバーポート、プロトコル、mDNS検出情報を表示します。ws://およびdns-sdのクイック接続スニペットを含みます。2秒ごとに自動更新されます。トレイメニューまたはコマンドパレットから開きます。

## 🌙 睡眠ステージング
30分以上のセッションの場合、履歴ビューには自動生成されたヒプノグラム — delta、theta、alpha、betaのバンドパワー比から分類された睡眠ステージ（Wake / N1 / N2 / N3 / REM）の階段状チャートが表示されます。履歴で長いセッションを展開すると、割合と時間を含むステージごとの内訳付きヒプノグラムが表示されます。注意: Museなどの民生用BCIヘッドセットは4つの乾式電極を使用するため、ステージングは概算です — 臨床的なポリソムノグラフではありません。

## ⚖  比較ウィンドウ
タイムライン上の任意の2つの時間範囲を選択し、平均バンドパワー分布、リラクゼーション/エンゲージメントスコア、前頭アルファ非対称性をサイドバイサイドで比較します。睡眠ステージング、詳細メトリクス、Brain Nebula™ — 高次元EEG空間で2つの期間がどれだけ類似しているかを示す3D UMAP投影を含みます。トレイメニューまたはコマンドパレット（⌘K → 比較）から開きます。

# オーバーレイとコマンドパレット
キーボードショートカットですべてのウィンドウで利用可能なクイックアクセスオーバーレイです。

## ⌨  コマンドパレット（⌘K / Ctrl+K）
アプリ内で実行可能なすべてのアクションを一覧表示するクイックアクセスドロップダウンです。入力を始めるとファジーフィルタリングされ、↑↓で移動、Enterで実行します。すべてのウィンドウで利用可能です。コマンドにはウィンドウを開く（設定、ヘルプ、検索、ラベル、履歴、キャリブレーション）、デバイスアクション（接続のリトライ、Bluetooth設定を開く）、ユーティリティ（ショートカットオーバーレイを表示、アップデートを確認）が含まれます。

## ?  キーボードショートカットオーバーレイ
任意のウィンドウで?を押すと（テキスト入力外）、すべてのキーボードショートカット — 設定 → ショートカットで設定されたグローバルショートカットに加え、⌘Kでコマンドパレット、⌘Enterでラベル送信などのアプリ内キー — を一覧表示するフローティングオーバーレイが切り替わります。再度?またはEscで閉じます。
