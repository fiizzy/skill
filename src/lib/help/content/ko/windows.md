# 창
{app}은 특정 작업을 위해 별도의 창을 사용합니다. 각 창은 트레이 컨텍스트 메뉴 또는 전역 키보드 단축키로 열 수 있습니다.

## 🏷  라벨 창
트레이 메뉴, 전역 단축키, 또는 메인 창의 태그 버튼으로 엽니다. 현재 EEG 순간에 주석을 달기 위한 자유 텍스트 라벨을 입력합니다(예: "명상", "집중 독서"). 라벨은 정확한 타임스탬프 범위와 함께 {dataDir}/labels.sqlite에 저장됩니다. Ctrl/⌘+Enter로 제출하거나 제출을 클릭합니다. Escape를 눌러 취소합니다.

## 🔍  검색 창
검색 창에는 세 가지 모드가 있습니다 — EEG 유사도, 텍스트, 인터랙티브 — 각각 다른 방식으로 기록된 데이터를 쿼리합니다.

## EEG 유사도 검색
시작/종료 날짜-시간 범위를 선택하고 해당 기간에 기록된 모든 EEG 임베딩에 대해 근사 최근접 이웃 검색을 실행합니다. HNSW 인덱스가 전체 기록에서 코사인 거리로 순위가 매겨진 가장 유사한 k개의 5초 EEG 에포크를 반환합니다. 낮은 거리 = 더 유사한 뇌 상태. 결과 타임스탬프와 겹치는 라벨이 인라인으로 표시됩니다. 참조 기간과 '느낌이 비슷한' 과거 순간을 찾는 데 유용합니다.

## 텍스트 임베딩 검색
개념, 활동, 또는 정신 상태를 자연어로 입력합니다(예: "깊은 집중", "불안함", "눈 감은 명상"). 쿼리는 라벨 인덱싱에 사용되는 것과 동일한 sentence-transformer 모델로 임베딩되고 HNSW 라벨 인덱스를 통한 코사인 유사도로 모든 주석과 매칭됩니다. 결과는 키워드 매칭이 아닌 의미적 유사도로 순위가 매겨진 사용자의 라벨입니다. 목록을 필터링하고 날짜 또는 유사도로 재정렬할 수 있습니다. 3D kNN 그래프가 이웃 구조를 시각화합니다: 쿼리 노드가 중앙에 위치하고 결과 라벨이 거리에 따라 바깥으로 방사됩니다.

## 인터랙티브 교차 모달 검색
자유 텍스트 개념을 입력하면 {app}이 4단계 교차 모달 파이프라인을 실행합니다: (1) 쿼리가 텍스트 벡터로 임베딩됩니다; (2) 의미적으로 가장 유사한 k개의 라벨이 검색됩니다(text-k); (3) 매칭된 각 라벨에 대해 평균 EEG 임베딩이 계산되어 일일 EEG HNSW 인덱스에서 가장 유사한 k개의 EEG 순간을 검색합니다(eeg-k); (4) 각 EEG 이웃에 대해 ±reach분 이내의 주석이 수집됩니다(label-k). 결과는 4개 노드 레이어를 가진 유향 그래프입니다 — 쿼리 → 텍스트 매치 → EEG 이웃 → 발견된 라벨 — 인터랙티브 3D 시각화로 렌더링되며 SVG 또는 Graphviz DOT로 내보낼 수 있습니다. text-k / eeg-k / label-k 슬라이더로 그래프 밀도를 제어하고, ±reach로 시간 검색 윈도우를 넓히거나 좁힙니다.

## 🎯  캘리브레이션 창
가이드 캘리브레이션 작업을 실행합니다: 설정 가능한 반복 횟수만큼 동작 단계를 번갈아 수행합니다(예: "눈 뜨기" → 휴식 → "눈 감기" → 휴식). 연결되어 스트리밍 중인 BCI 디바이스가 필요합니다. 캘리브레이션 이벤트는 Tauri 이벤트 버스와 WebSocket을 통해 전달되므로 외부 도구가 동기화할 수 있습니다. 마지막 완료된 캘리브레이션의 타임스탬프가 설정에 저장됩니다.

## ⚙  설정 창
네 개의 탭: 설정, 단축키(전역 핫키, 명령 팔레트, 앱 내 키), EEG 모델(인코더 및 HNSW 상태). 트레이 메뉴 또는 메인 창의 기어 버튼으로 엽니다.

## ?  도움말 창
이 창입니다. {app} 인터페이스의 모든 부분에 대한 완전한 참조 — 메인 대시보드, 각 설정 탭, 모든 팝업 창, 트레이 아이콘, WebSocket API. 트레이 메뉴에서 엽니다.

## 🧭  설정 마법사
Bluetooth 페어링, 헤드셋 착용, 첫 캘리브레이션을 안내하는 5단계 초기 실행 마법사입니다. 첫 실행 시 자동으로 열립니다. 명령 팔레트(⌘K → Setup Wizard)에서 언제든지 다시 열 수 있습니다.

## 🌐  API 상태 창
현재 연결된 모든 WebSocket 클라이언트와 스크롤 가능한 요청 로그를 보여주는 실시간 대시보드입니다. 서버 포트, 프로토콜, mDNS 검색 정보를 표시합니다. ws:// 및 dns-sd용 빠른 연결 스니펫이 포함됩니다. 2초마다 자동 새로고침됩니다. 트레이 메뉴 또는 명령 팔레트에서 엽니다.

## 🌙 수면 단계 분류
30분 이상 지속되는 세션의 경우 히스토리 뷰에 자동 생성된 히프노그램이 표시됩니다 — delta, theta, alpha, beta 대역 전력 비율로 분류된 수면 단계(Wake / N1 / N2 / N3 / REM)의 계단식 차트입니다. 히스토리에서 긴 세션을 확장하면 비율과 지속 시간을 포함한 단계별 분석과 함께 히프노그램을 볼 수 있습니다. 참고: Muse와 같은 소비자 BCI 헤드셋은 4개의 건식 전극을 사용하므로 단계 분류는 대략적입니다 — 임상 수면 다원검사가 아닙니다.

## ⚖  비교 창
타임라인에서 두 시간 범위를 선택하여 평균 대역 전력 분포, 이완/집중 점수, 전두엽 알파 비대칭을 나란히 비교합니다. 수면 단계 분류, 고급 지표, Brain Nebula™ — 두 기간이 고차원 EEG 공간에서 얼마나 유사한지를 보여주는 3D UMAP 프로젝션이 포함됩니다. 트레이 메뉴 또는 명령 팔레트(⌘K → Compare)에서 엽니다.

# 오버레이 및 명령 팔레트
키보드 단축키를 통해 모든 창에서 사용할 수 있는 빠른 접근 오버레이입니다.

## ⌨  명령 팔레트 (⌘K / Ctrl+K)
앱의 모든 실행 가능한 동작을 나열하는 빠른 접근 드롭다운입니다. 입력하여 퍼지 필터링하고, ↑↓로 탐색하고, Enter로 실행합니다. 모든 창에서 사용 가능합니다. 명령에는 창 열기(설정, 도움말, 검색, 라벨, 히스토리, 캘리브레이션), 디바이스 동작(재연결 시도, Bluetooth 설정 열기), 유틸리티(단축키 오버레이 표시, 업데이트 확인)가 포함됩니다.

## ?  키보드 단축키 오버레이
아무 창에서나(텍스트 입력 외) ?를 누르면 모든 키보드 단축키를 나열하는 플로팅 오버레이가 전환됩니다 — 설정 → 단축키에서 구성된 전역 단축키와 명령 팔레트의 ⌘K, 라벨 제출의 ⌘Enter 등 앱 내 키가 포함됩니다. ?를 다시 누르거나 Esc로 닫습니다.
