# 메인 창
메인 창은 기본 대시보드입니다. 실시간 EEG 데이터, 디바이스 상태, 신호 품질을 표시합니다. 메뉴 바에서 항상 볼 수 있습니다.

## 상태 히어로
상단 카드는 BCI 디바이스의 실시간 연결 상태를 표시합니다. 색상 링과 배지로 디바이스의 연결 해제, 스캔 중, 연결됨 또는 Bluetooth 꺼짐 상태를 나타냅니다. 연결되면 디바이스 이름, 시리얼 번호, MAC 주소가 표시됩니다(클릭하여 표시/숨기기).

## 배터리
연결된 BCI 헤드셋의 현재 배터리 충전량을 나타내는 진행률 표시줄입니다. 충전량이 감소함에 따라 색상이 녹색(높음)에서 주황색을 거쳐 빨간색(낮음)으로 변합니다.

## 신호 품질
4개의 색상 코드 점 — 각 EEG 전극(TP9, AF7, AF8, TP10)에 하나씩. 녹색 = 양호한 피부 접촉 및 낮은 노이즈. 노란색 = 보통(일부 아티팩트). 빨간색 = 불량(높은 노이즈 / 전극 느슨함). 회색 = 신호 없음. 품질은 원시 EEG 데이터의 롤링 RMS 윈도우로 계산됩니다.

## EEG 채널 그리드
각 채널의 최신 샘플 값(µV 단위)을 표시하는 4개의 카드로, 아래 파형 차트와 색상이 일치합니다.

## 가동 시간 및 샘플
가동 시간은 현재 세션이 시작된 이후의 경과 시간(초)을 계산합니다. 샘플은 이 세션에서 헤드셋으로부터 수신한 원시 EEG 샘플의 총 수입니다.

## CSV 녹화
연결되면 REC 표시기가 {dataDir}/에 기록 중인 CSV 파일명을 표시합니다. 원시(필터링되지 않은) EEG 샘플은 지속적으로 저장됩니다 — 세션당 하나의 파일입니다.

## 대역 전력
각 표준 EEG 주파수 대역의 상대 전력을 보여주는 실시간 막대 차트입니다: Delta(1–4 Hz), Theta(4–8 Hz), Alpha(8–13 Hz), Beta(13–30 Hz), Gamma(30–50 Hz). 512 샘플 Hann 윈도우 FFT에서 ~4 Hz로 업데이트됩니다. 각 채널이 개별적으로 표시됩니다.

## 전두엽 알파 비대칭(FAA)
실시간 전두엽 알파 비대칭 지수를 보여주는 중앙 앵커 게이지입니다: ln(AF8 α) − ln(AF7 α). 양수 값은 우측 전두엽 알파 전력이 더 크다는 것을 나타내며, 이는 좌반구 접근 동기와 관련이 있습니다. 음수 값은 회피 경향을 나타냅니다. 값은 지수 이동 평균으로 평활화되며 일반적으로 −1에서 +1 범위입니다. FAA는 eeg.sqlite의 모든 5초 임베딩 에포크와 함께 저장됩니다.

## EEG 파형
모든 채널의 필터링된 EEG 신호를 스크롤하는 시간 도메인 차트입니다. 각 파형 아래에는 시간에 따른 주파수 내용을 보여주는 스펙트로그램 테이프가 있습니다. 차트는 가장 최근 ~4초의 데이터를 표시합니다.

## GPU 사용률
메인 창 상단에 있는 작은 차트로, GPU 인코더 및 디코더 사용률을 표시합니다. EEG 임베딩 인코더가 활성 상태일 때만 표시됩니다. wgpu 파이프라인이 실행 중인지 확인하는 데 유용합니다.

# 트레이 아이콘 상태

## 회색 — 연결 해제
Bluetooth가 켜져 있으나 BCI 디바이스가 연결되지 않았습니다.

## 주황색 — 스캔 중
BCI 디바이스를 검색하거나 연결을 시도하고 있습니다.

## 녹색 — 연결됨
BCI 디바이스에서 실시간 EEG 데이터를 스트리밍하고 있습니다.

## 빨간색 — Bluetooth 꺼짐
Bluetooth 라디오가 꺼져 있습니다. 스캔이나 연결이 불가능합니다.
