### Features

- **Interactive graph screenshots**: Added screenshot nodes to the interactive 3D cross-modal search graph. When the "Screenshots" checkbox is enabled, nearby screenshots (±120 s) are fetched for each EEG-point node and injected as proper connected graph nodes (cyan spheres with thumbnail sprites). Screenshots participate fully in the graph — they are positioned near their parent EEG node, connected by `screenshot_link` edges, and respond to selection highlighting like any other node. The toggle is available in the interactive mode header row alongside the existing Deduplicate and PCA cluster options.
